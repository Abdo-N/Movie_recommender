# Movie Recommender System

A project to practice pandas, SQL, and git by building a movie recommendation system.

## Dataset
Using MovieLens 100K dataset from GroupLens Research.

## How it works
1. used pandas to view and clean data
2. loaded cleaned and edited data to SQLite data base
3. allowed access between code and data base with pandas through SQL query

## Usability
1. User inputs movie name
2. algorithm finds users that have rated the movie highly (4+ stars)
3. Then checks: What other movies did the users rate highly?
4. Returns: Those movies as recommendations
